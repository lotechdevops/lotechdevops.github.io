# Sobre mim

Olá, me chamo Neto e atuo como DevOps desde 2020.

Tenho experiência na construção e manutenção de ambientes em nuvem e on-premises, utilizando práticas de infraestrutura como código com Terraform, Ansible e Packer.

Trabalho com foco em Kubernetes, provisionamento de ambientes, pipelines de CI/CD, segurança com HashiCorp Vault e ferramentas de observabilidade como Prometheus, Grafana e Alertmanager.

Possuo conhecimento em desenvolvimento de scripts de automação utilizando Python e Shell Script, além de sólida experiência em sistemas Linux.

Este espaço no GitHub é onde compartilho alguns dos meus estudos, projetos pessoais e anotações técnicas.

## Temas que abordo:

- Kubernetes
- AWS
- Terraform
- Ansible
- Automação
- Observabilidade
- Infraestrutura como Código
